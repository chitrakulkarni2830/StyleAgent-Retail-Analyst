STEP 2 â€” INSTALL OLLAMA (the local AI brain)

Ollama runs a free AI model (called llama3) on your own computer.
No internet needed, no account needed, no API key ever.

1. Go to https://ollama.com and click "Download for Mac".
2. Open the downloaded file and drag Ollama to your Applications.
3. Open Ollama from your Applications folder. You will see a

4. Now open Terminal and type this, then press Enter:
      ollama pull llama3
5. This will download the llama3 AI model (about 4GB).
   Wait for it to finish. You only do this once.
6. Once done, type this to check it works:
      ollama run llama3 "Say hello"
   You should see a response from the AI. Press Ctrl+D to exit.

TROUBLESHOOTING
----------------------------------------------------------------

Q: The app opens but shows an error about "ModuleNotFoundError"
A: You missed Step 4. Run: pip install -r requirements.txt

Q: The Generate button does nothing after a few seconds
A: Make sure Ollama is running. Open the Ollama app from
   Applications, then try again.

Q: I see "database/inventory.db not found"
A: You missed Step 5. Run: python3 database/setup_database.py

Q: The window looks plain grey instead of dark
A: This is a Tkinter display issue. Run: brew install python-tk
   Then try again.
